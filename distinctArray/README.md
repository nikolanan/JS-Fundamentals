You will be given an array of integer numbers on the first line of the input.
Remove all repeating elements from the array.
Print the result elements separated by a single space


Sample input:

[1, 2, 3, 4] 

Sample output:

1 2 3 4

Sample input:

[7, 8, 9, 7, 2, 3,
4, 1, 2]

Sample output:

7 8 9 2 3 4 1
