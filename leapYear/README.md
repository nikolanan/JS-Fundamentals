Write a JS function to check whether a year is a leap. Leap years are either divisible by 4 but not by 100 or are
divisible by 400. The output should be following:

• If the year is a leap, print: "yes"

• Otherwise, print: "no"

Sample input:
1984

Sample output:
yes

Sample input:
2003

Sample ouput:
no

Sample input:
4

Sample output:
no
